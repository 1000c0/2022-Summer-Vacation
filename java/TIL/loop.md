# while 문
```java
int treehit = 0;
while(treehit < 10){
    treehit++;
    System.out.println("나무를" + treehit + "번 찍었습니다.")
    if(treehit == 10){
        System.out.print("나무가 넘어갑니다")
    }
}
```

# break와 continue
- break : break를 만나면 while문을 빠져나간다
- continue : continue를 만나면 조건식으로 돌아간다
